# 🚀 QUICK START GUIDE

## Installation (30 seconds)

```bash
# Navigate to project directory
cd c:\Users\bhara\OneDrive\Desktop\Loan_Ai

# Install dependencies
pip install -r requirements.txt
```

## Run the App (5 seconds)

```bash
streamlit run app.py
```

## That's it! 🎉

The app will open automatically in your browser at: http://localhost:8501

## Test the App

1. Upload any PDF file (financial statements, reports, etc.)
2. Click "Analyze Documents"
3. View extracted text and download results

## Troubleshooting

**Issue**: Module not found
**Fix**: `pip install -r requirements.txt`

**Issue**: Port already in use
**Fix**: `streamlit run app.py --server.port 8502`

**Issue**: PDF extraction fails
**Fix**: Ensure PDF is not password-protected or corrupted
